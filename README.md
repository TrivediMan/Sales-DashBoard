📊 Sales Analytics Dashboard

An interactive Sales Analytics Dashboard built to visualize key business metrics such as total sales, return rates, regional performance, and product insights. This dashboard helps stakeholders quickly understand trends and make data-driven decisions.

🚀 Features
📈 Total Sales Overview
Displays overall sales performance (e.g., 844.02K)
🔄 Return Rate Tracking
Monitor product return percentage
🎯 Goal Tracking
Visual comparison of actual vs target sales
🏆 Top Performers
Top category (e.g., Office Supplies)
Top region (e.g., South)
📦 Product-Level Insights
Sales by product
Units sold
Price analysis
🌍 Regional Analysis
Sales distribution by region
Customer distribution by region
📊 Category Breakdown
Office Supplies
Furniture
Technology
📅 Yearly Sales Trends
Compare performance across years
🎛️ Interactive Filters
Filter by category (Accessories, Chairs, Phones, etc.)
Filter by region (East, West, North, South)
